FAQ

Common questions

What is vibe coding?

Vibe coding is directing AI to write code for you. Instead of typing code yourself, you describe what you want, review AI output, and guide iterations until your app works perfectly.

Do I need coding experience?

No! You'll learn to communicate with AI coding tools effectively. The AI writes the code - you control the vision and make decisions.

What tools will I learn?

You'll master AI coding assistants like Claude, Cursor, and other AI-powered development tools that let you build real applications through conversation.
